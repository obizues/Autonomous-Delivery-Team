# test_autonomous_delivery_repo

A minimal Python project with a web UI for submitting feature requests and viewing before/after changes. Designed for easy extension and integration with autonomous delivery workflows.

## Features
- Submit feature requests via web UI
- View before/after changes
- Extensible structure for workflow integration

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app/main.py`

## Structure
- `app/` - Web UI (Streamlit)
- `autonomous_delivery/` - Core logic and workflow integration
- `tests/` - Unit and integration tests
